To use: 	1. Open project in Eclipse
		2. Turn the DRTHLobject's lToggle static variable on or off depending on whether or not you want DRT-h or DRT-hL
		3. (Optional) Add your project as a package, and implement tests like the tests in our Example. Be sure to change the package name in drthl.java if you're doing this.
		3. Run drthl from the run menu.
		4. Observe failures and results in the console.
